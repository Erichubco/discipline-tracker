# Discipline · Eric

Tracker quotidien de discipline et routine (tâches du jour, historique 7 jours, poids, lecture). App statique, aucune donnée n'est envoyée à un serveur — tout est stocké en `localStorage` sur l'appareil.

## Installer sur iPhone

1. Ouvrir l'URL du site dans Safari
2. Bouton Partager (carré + flèche vers le haut)
3. "Sur l'écran d'accueil"

L'app s'ouvre alors en plein écran comme une app native, avec l'icône Discipline.

## Dev local

Fichier unique `index.html`, pas de build. Ouvrir directement dans un navigateur, ou servir en local :

```bash
python3 -m http.server 8000
```
